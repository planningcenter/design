# Tab Header

> Header grid navigating a resource and its relationships

## Selectors

| data-attribute                  | usage         |
| ------------------------------- | ------------- |
| `data-tab-header--container`    | flex template |
| `data-tab-header--tab-group`    |               |
| `data-tab-header--tab`          |               |
| `data-tab-header--tab="target"` |               |
| `data-tab-header--action-group` |               |
| `data-tab-header--action`       |               |

export function ExampleContainer({ style, ...props }) {
  return (
    <div
      style={{ backgroundColor: "tomato", paddingTop: "1rem", ...style }}
      {...props}
    />
  );
}

## Disabled tab

`data-tab-header--tab` elements without an `href` attribute get `disabled` treatment.

The effect is identical between an `a` (with no `href` attr) and a `span`.
